How to filter out events from being used by the Coveo Machine Learning models?

To filter out events from being used by the Coveo Machine Learning models, you can create a filter rule in the Coveo Cloud administration console. 1. Login to the Coveo Cloud administration console. 2. Select the organization in which you want to create the filter rule. 3. Click on the "Machine Learning" tab. 4. Click on "Filter Rules" in the left-hand menu. 5. Click on "Add Filter Rule". 6. Choose the type of filter you want to create (e.g. exclude specific events). 7. Define the criteria for the filter (e.g. event type, user, etc.). 8. Save the filter rule. Once the filter rule is saved, the specified events will be excluded from being used by the Coveo Machine Learning models.
